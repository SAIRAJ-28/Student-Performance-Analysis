                                           🎓 Student Performance Analyzer
📊 Project Overview

The Student Performance Analyzer is a data analytics project that examines student academic performance to identify patterns and factors influencing results. The analysis focuses on metrics such as attendance, study hours, subject performance, and pass/fail rates.
The objective of this project is to demonstrate data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard visualization using industry-relevant tools.

🎯 Project Objectives

1. Analyze student academic data to identify performance trends

2. Evaluate the impact of attendance and study hours on student marks

3. Identify subjects with higher failure rates

4. Compare department-wise student performance

5. Build an interactive Power BI dashboard for insights

🛠 Tools & Technologies

Python – Data analysis and visualization

Pandas / Matplotlib – Exploratory Data Analysis (EDA)

SQL – Data querying and aggregation

Power BI – Interactive dashboard and data visualization

Excel / CSV – Data storage and preprocessing

🔍 Key Insights

Students with attendance above 75% generally achieve higher marks.
Some subjects show higher failure rates, indicating potential learning challenges.
Increased study hours correlate positively with performance.
Performance varies across departments, suggesting targeted academic support may be required.
